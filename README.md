# CleanResponse 插件

> 自动清除 LLM 回复中括号内的“心声”、“动作”或“神态描写”，让文本更干净，更适合 TTS 朗读。

## ✨ 功能说明

许多大语言模型（LLM）在生成回复时会包含类似以下内容：

- `(思考中...)`
- `（微笑着看向你）`
- `*轻声说道*`
- `[系统提示：...]`

这些内容对人类阅读可能有帮助，但对语音合成（TTS）来说会造成干扰（比如读出“左括号思考中右括号”）还有就是我个人会觉得他还是AI不像人。

本插件会在 LLM 生成结果后 自动移除所有中文/英文括号及其内部内容，确保输出纯文本。

示例：
输入：  “（开心地跳起来）今天天气真好！”
输出：  “今天天气真好！”

## ⚙️ 配置方式

无配置。

- 默认开启
